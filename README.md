# External Example Base

External root чит для standoff 2 на C++23.

## Стек

- C++23
- Android NDK (r27d)
- ImGui (Dear ImGui)

## Roadmap

### Этап 1

- [x] Настройка среды сборки.
- [x] Инициализация прозрачного оверлея (Native Activity / SurfaceFlinger).
- [x] Интеграция ImGui и базовая обработка сенсорного ввода.
- [ ] Карсивый интерфейс как у премиального зенина за 2к рублей.
- [ ] Реализация модуля безопасного чтения/записи памяти процесса (Memory Manager).

### Этап 2

- [ ] Поиск стабильных смещений (Offsets).
- [ ] Определение базовых структур данных.
- [ ] Реализация функции проецирования координат WorldToScreen (W2S).

### Этап 3

- [ ] Отрисовка 2D-боксов и линий направления (Snaplines).
- [ ] Чтение костей игрока для скелетного ESP.

### Этап 4

- [ ] Математический расчет углов (Vector Math / Angle Calculation).
- [ ] Плавная доводка прицела (Smooth) и ограничение по углу обзора (FOV Check).
- [ ] Компенсация отдачи оружия (RCS).

### Этап 5

- [ ] Парсер и менеджер конфигураций (JSON/INI).
- [ ] Система кастомных тем для ImGui меню.

## Cборка

### Требования к окружению

- Android NDK (r27d)

## Совместная разработка (Contribution)

- Изменения в структурах игры и новые офсеты добавляйте строго в src/sdk/offsets.hpp с указанием даты обновления/версии игры.
