# external_example_base — документация

Внешний (External) root-чит для **Standoff 2 (v0.39.1)** на языке C++.  
Проект ориентирован на работу через привилегии суперпользователя (Root) для чтения/записи памяти процесса и отрисовки оверлея поверх игры.

---

## 🛠 Технический стек
* **Язык программирования:** C++23
* **Сборочная система:** Android NDK
* **Интерфейс:** Dear ImGui

---

## 💻 Руководство по сборке (для разработчиков)

### Требования к окружению
* Android NDK (рекомендуемая версия r27)
* Установленный `adb` для отладки

---

## 🛠 Отладка и Логирование

В проекте используется макрос отладки, выводящий сообщения в системный лог Android (`logcat`).

Для отслеживания логов во время работы приложения используйте команду:
```bash
adb logcat -s external_example_base:V
```

## 🤝 Соглашение о совместной разработке (Contribution Guide)
* **Смещения (Offsets):** Изменения в структурах игры и новые офсеты добавляйте строго в `src/sdk/offsets.hpp` с указанием даты обновления.
